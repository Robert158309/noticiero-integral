---

## 📌 About the project

This is a web-based news-style landing page developed for a seminar project.  
The topic focuses on **corruption and its impact on Economic, Social and Cultural Rights (DESCA)** in Guatemala.

The goal of this project is to present research findings in a creative and interactive way, simulating a digital news platform.

---

## 🎯 Purpose

- Present research in a visual and interactive format  
- Simulate a professional news portal  
- Communicate the effects of corruption in society  
- Develop web design and programming skills  

---

## 🛠️ Technologies used

- HTML5  
- CSS3  
- JavaScript (Vanilla)
